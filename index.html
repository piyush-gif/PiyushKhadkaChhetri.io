Data cleaning in SQL

This SQL script cleans and transforms Nashville housing data for better analysis. It standardizes dates, fills missing property addresses, and splits addresses into separate columns (address, city, state). It also normalizes categorical values (e.g., 'Y' → 'Yes', 'N' → 'No') and removes duplicates using ROW_NUMBER(). Finally, it drops unnecessary columns, improving data quality for analysis.

Pneumonia Recognition using Monai(PRM)

The "Pneumonia Recognition using MONAI" project aims to detect pneumonia from chest X-ray images by leveraging deep learning techniques within the MONAI (Medical Open Network for AI) framework. The system employs a custom convolutional neural network (CNN) model and achieves an accuracy of 80%. A Flask-based web interface allows users to upload images and receive diagnostic results, offering an efficient tool for medical diagnostics.
